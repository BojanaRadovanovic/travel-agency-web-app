# travel-agency-web-app

This project is a functional travel agency web application developed using PHP and Bootstrap.

## Features
- Responsive design for all devices
- Display of travel offers and destinations
- User-friendly navigation
- Dynamic content using PHP

## Technologies
- HTML
- CSS
- Bootstrap
- PHP

## Description
The website represents a travel agency platform where users can explore different travel destinations and offers. The project focuses on combining front-end design with back-end functionality using PHP.

## Learning Outcome
This project helped me improve my web development skills, especially in building dynamic web applications and working with responsive design.

## Setup Instructions
To run this project properly, you need to install and include Font Awesome:

- Download Font Awesome (version 6.7.2)
- Extract the folder **fontawesome-free-6.7.2-web**
- Place it inside your project directory
- Make sure it is properly linked in the HTML files
